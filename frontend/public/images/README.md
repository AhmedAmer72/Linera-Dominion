# Image Assets for Linera Dominion

Save your generated images to these locations:

## Logo
- `logo.png` - Main game logo (the hexagonal crown with spaceship)

## Resource Icons  
Save in `/resources/` folder:
- `iron.png` - Iron Ore Crystal (orange glowing crystal)
- `deuterium.png` - Blue Deuterium Flask
- `crystals.png` - Purple Crystal Gem

## Building Icons
Save in `/buildings/` folder:
- `plasma-mine.png` - Plasma Mine
- `ore-processor.png` - Ore Processor  
- `research-lab.png` - Research Lab
- `shipyard.png` - Shipyard (Star Forge)
- `defense-turret.png` - Defense Turret

## Ship Icons
Save in `/ships/` folder:
- `scout.png` - Scout Fighter
- `battlecruiser.png` - Battlecruiser
- `carrier.png` - Carrier
- `dreadnought.png` - Dreadnought

## Research Icons
Save in `/research/` folder:
- `genetics.png` - DNA Helix (Genetics research)
- `physics.png` - Atom (Physics research)
- `propulsion.png` - Rocket Engine (Propulsion research)
- `shields.png` - Shield Dome (Shield research)

## Image Requirements
- Format: PNG with transparent background
- Recommended size: 256x256 or larger
- Style: Consistent sci-fi holographic theme
